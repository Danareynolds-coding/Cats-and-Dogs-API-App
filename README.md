# Cats and Dogs API App
 ### This is an App with 2 pages one for cat breeds and one for dog breeds:smile::+1:

#### This App allows you to pull information on cat breeds from the Cat API and Dog API both are published by **ThatApiCompany* 

  * Page One: Form that pulls up Cat:cat: by Breed. Provides breed discription such as weight temperment, origin, life span and an image.
  * Page Two: Form that pulls up Dog:dog: by Breed. Provides breed discription such as breed group, purpose bred for, temperment, origin, life span and an image.

### Instructions to User:
  * Page One is a form page to pull up cats by breed.  Simply type the breed you are wanting information on in the form and hit enter..........
  * Page Two is also a form page to bring up dogs by breeds. The form works the same as the cat form...........
